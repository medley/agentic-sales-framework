# Input Directory

Drop your source files here. This directory is **read-only** - skills will never modify these files.

## Subdirectories

| Directory | What to put here |
|-----------|-----------------|
| `battlecards/` | Competitive intelligence, win/loss analysis |
| `call_transcripts/` | Meeting transcripts, call recordings |
| `email_threads/` | Email conversations for deal context |
| `personas/` | Buyer persona documents, ICP definitions |
| `playbooks/methodologies/` | Sales methodology source materials |
| `product_docs/` | Product documentation, datasheets, pricing |

## Supported Formats

- PDF, DOCX, PPTX (converted via document conversion skills)
- Markdown files (used directly)
- Plain text files
